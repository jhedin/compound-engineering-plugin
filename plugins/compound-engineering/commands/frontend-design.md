# Frontend Design

Create distinctive, production-grade frontend interfaces with high design quality.

## Input

<design_request> #$ARGUMENTS </design_request>

## Instructions

Follow the frontend-design skill to create:

- **Distinctive visuals** - Avoid generic AI aesthetics
- **Production-ready code** - Clean HTML/CSS/JS or React
- **Responsive design** - Mobile-first approach
- **Accessibility** - WCAG compliance
- **Performance** - Optimized assets and loading

## Key Principles

- Start with a clear visual concept
- Use a consistent design system
- Pay attention to spacing, typography, and color
- Add subtle animations and interactions
- Test across browsers and devices

## Example

```
/frontend-design a pricing page with three tiers
/frontend-design hero section for a developer tools landing page
```
